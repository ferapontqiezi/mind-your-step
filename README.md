# mind-your-step（一步两步）
mind-your-step（也叫一步两步）是基于cocos游戏引擎的官方教程初步尝试的小游戏，入门者的第一次尝试。

## Get started

环境：

- Creator3D 1.1.0

## Run it

打开Creator点击浏览器运行即可进入游戏。

## Conclusion

~~做游戏果然是动画最难。~~
